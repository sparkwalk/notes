# Linux系统环境变量

## 1.头文件的搜索路径:

```  bash
export C_INCLUDE_PATH=<your include path>
```

## 2.编译链接库搜索路径

``` bash
export LIBRARY_PATH=<your lib path>
```

## 3.运行时库搜索路径

``` bash
export LD_LIBRARY_PATH=<your ldlib path>
```

## 4.C++ 头文件的搜索路径

``` bash
export CPLUS_INCLUDE_PATH=<your c++ include path>
```
